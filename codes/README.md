# AroMIP — Training Code

[![License: MIT](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)](../LICENSE)

> This folder contains the training and testing code that produced the per-residue weights used by the [AroMIP web tool](https://fidhanazreen.github.io/AroMIP/).

---

## Overview

Intrinsically disordered proteins (IDPs) frequently interact with cellular membranes through short hydrophobic motifs centered on aromatic residues (Phe, Trp, Tyr). AroMIP learns a **multiplicative scoring model** from structural data generated by the PPM3/OPM energy-minimisation tool. Each flanking amino acid in a 9-residue aromatic-centred window is assigned a weight *α* reflecting how strongly it promotes or suppresses membrane insertion.

---

## Scoring Model

### 1. Fragment Scanning
Every **9-residue window** whose **center residue (position 5)** is **F**, **W**, or **Y** is extracted from the input sequence.

### 2. Multiplicative Scoring
Each flanking residue (positions 1–4 and 6–9) contributes a multiplicative factor. The raw insertion propensity *Q* is:

$$Q = \prod_{\substack{i=1 \\ i \neq 5}}^{9} \alpha_i^{(\text{AA}_i)}$$

where $\alpha_i^{(\text{AA}_i)}$ is the trained weight for the amino acid at position *i*, specific to the central aromatic (F, W, or Y).

### 3. Generalized Sigmoid
*Q* is converted to a membrane-insertion propensity score *P* ∈ [0, 1) via a generalized sigmoid:

$$P = \begin{cases} \dfrac{Q}{1 + Q} & \text{if } Q > 0 \\ 0 & \text{otherwise} \end{cases}$$

### 4. Classification

| Score *P* | Prediction     |
|-----------|----------------|
| > 0.5     | **Inserter**   |
| ≤ 0.5     | Non-inserter   |

Parameters are optimised separately for **F**, **W**, and **Y** using `scipy.optimize.least_squares` (Trust-Region Reflective method).

---

## Binary Labelling

Sequences are labelled from PPM3 insertion depth scores using a dynamic threshold:

- Default cutoff: depth score ≥ 40 → **Inserter** (label = 1)
- Each **R** in the flanking positions lowers the threshold by 15
- Each **K** lowers it by 10
- Minimum threshold: 9

This accounts for the membrane-facilitating effect of positively-charged residues.

## Datasets

### 5letter — Synthetic combinatorial dataset

The middle position is fixed as F, W, or Y. Each of the 8 context positions is drawn from a reduced alphabet **{L, R, G, N, E}** (5 amino acids), giving **5⁸ = 390 625** sequences per aromatic — full combinatorial coverage. Structural insertion data were generated by PPM3.

The training scripts run a **convergence study**, repeating optimisation at increasing dataset sizes (1 000 → 10 000 → 100 000 → full) to confirm that the learned weights are stable.

### HumanProteomeIDRs — Human proteome IDR dataset

9-residue windows extracted from human **intrinsically disordered regions** (IDRs), centred on F, W, or Y residues. The flanking positions span the full **20 amino acid** alphabet. The dataset is pre-split 80 / 20 into training and test sets. This dataset directly reflects the biological context for which AroMIP is designed.

---

## Data Format

All `.csv` files are **whitespace-separated with no header**:

| Column | Content |
|--------|---------|
| 0 | 9-residue sequence string |
| 1 | PPM3 intermediate value (not used) |
| 2 | PPM3 intermediate value (not used) |
| 3 | PPM3 membrane insertion depth score |

Sequences containing non-standard amino acid `X` are excluded before training.

---

## Amino-acid Groups (HumanProteomeIDRs)

Residues with strong positive weights (> 1) in the flanking positions — such as Ile, Leu, Val, Met, and other aromatics — generally promote membrane insertion, while charged and polar residues (Asp, Glu, Asn, Gln) suppress it.

| Group | Residues | Property |
|-------|----------|----------|
| group1 | V L I M F W Y | Hydrophobic / aromatic — highest *α* |
| group2 | G P C A | Helix-breaker / small apolar |
| group3 | S T N Q H | Polar uncharged |
| group4 | K R | Positively charged |
| group5 | D E | Negatively charged — lowest *α* |

## Usage

### 5letter

```bash
cd 5letter
python train_F.py   # Phenylalanine
python train_W.py   # Tryptophan
python train_Y.py   # Tyrosine
```

Each script saves a parameter convergence table and log-scale plot to `results/`.

### HumanProteomeIDRs

```bash
cd HumanProteomeIDRs
python train_test_F.py   # Phenylalanine
python train_test_W.py   # Tryptophan
python train_test_Y.py   # Tyrosine
```

Each script prints training accuracy, test accuracy, confusion matrix (TP / TN / FP / FN), and the top 10 highest-scoring test sequences. Trained weights are saved to `results/`.
