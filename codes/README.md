# AroMIP – Aromatic amino acid Membrane Insertion Prediction

Predicts whether a Phenylalanine (F), Tryptophan (W), or Tyrosine (Y) residue at the centre of a 9-residue peptide window will insert into a lipid membrane.

---

## Background

Aromatic residues are frequently found at the membrane–water interface of transmembrane and peripheral membrane proteins. Their insertion is influenced by the identity of the surrounding (context) residues. This model learns a per-amino-acid **q parameter** from structural data generated by the PPM3/OPM energy-minimisation tool and uses it to predict membrane insertion for new sequences.

---

## Model

For a 9-residue window `s₁ s₂ s₃ s₄ [F/W/Y] s₆ s₇ s₈ s₉`, the insertion probability is:

```
Q      = q[s₁] × q[s₂] × q[s₃] × q[s₄] × q[s₆] × q[s₇] × q[s₈] × q[s₉]
P(ins) = 1 / (1 + (Q_th / Q)^n)      Q_th = 1, n = 1 (fixed)
```

Each context amino acid contributes a multiplicative factor `q`. A high product Q means the surrounding residues collectively promote insertion; a low Q means they oppose it.

**Binary labelling** uses a dynamic depth threshold from PPM3:
- Default cutoff: depth score ≥ 40 → inserted (label = 1)
- Sequences with R/K residues: threshold lowered by 15 per R and 10 per K (minimum 9), because positively-charged residues facilitate membrane association.

**Optimisation**: `scipy.optimize.least_squares` with the Trust-Region Reflective (TRF) method minimises the sum of squared residuals between predicted probabilities and binary labels.

## Data format

All `.csv` files are **whitespace-separated with no header**:

| Column | Content |
|--------|---------|
| 0 | 9-residue sequence string |
| 1 | PPM3 intermediate value (not used) |
| 2 | PPM3 intermediate value (not used) |
| 3 | PPM3 membrane insertion depth score (used for labelling) |

Sequences containing non-standard amino acid `X` are excluded before training.

---

## Datasets

### 5letter
Fully enumerated synthetic 9-mers: the middle position is fixed as F, W, or Y, and each of the 8 context positions is drawn from a reduced alphabet **{L, R, G, N, E}** (5 amino acids). This gives 5⁸ = 390 625 sequences per aromatic, covering the full combinatorial space. Structural data were generated by running each sequence through PPM3.

### HumanIDR
Windows extracted from **human Intrinsically Disordered Regions** (IDRs). Each window is centred on an F, W, or Y residue found in a region predicted to be disordered. The surrounding residues are drawn from the full 20-AA alphabet. The dataset is pre-split 80/20 into training and test sets.

## Usage

### 5letter – training convergence study

```bash
cd 5letter
python train_F.py   # Phenylalanine
python train_W.py   # Tryptophan
python train_Y.py   # Tyrosine
```

Each script:
1. Loads and labels the full dataset.
2. Runs optimisation at training sizes 1 000, 10 000, 100 000, and the full dataset.
3. Saves a CSV of per-AA q parameters at each training size to `results/`.
4. Saves a log-scale convergence plot to `results/`.

### HumanIDR – train and test

```bash
cd HumanIDR
python train_test_F.py   # Phenylalanine
python train_test_W.py   # Tryptophan
python train_test_Y.py   # Tyrosine
```

Each script:
1. Loads and labels the training set, trains the 20-AA model.
2. Prints training accuracy and confusion matrix (TP/TN/FP/FN).
3. Saves trained q parameters to `results/`.
4. Loads the test set, evaluates, and prints test accuracy and confusion matrix.
5. Prints the top 10 highest-scoring test sequences.

## Key parameters

| Parameter | Value | Description |
|-----------|-------|-------------|
| `MIDDLE_POS` | 5 | 1-indexed position of the fixed aromatic residue |
| `Q_TH_FIXED` | 1.0 | Sigmoid midpoint (Q_th); kept fixed during optimisation |
| `N_FIXED` | 1.0 | Sigmoid steepness (n); kept fixed during optimisation |
| `BASE_THRESHOLD` | 40 | Default PPM3 depth cutoff for positive labelling |
| `MIN_THRESHOLD` | 9 | Minimum depth cutoff (for heavily R/K-charged sequences) |
| `R_WEIGHT` | 15 | Threshold reduction per R residue |
| `K_WEIGHT` | 10 | Threshold reduction per K residue |
| `SCALE_FACTOR` | (10⁶)^(1/8) ≈ 7.5 | Parameter scaling for F and W (5letter and IDR) |
| `SCALE_FACTOR` | (10¹)^(1/8) ≈ 1.33 | Parameter scaling for Y (lower insertion propensity) |
